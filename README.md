This code implements several generative AI models for image generation, including a VAE, DDPM, DDIM, and latent diffusion model. It defines a variance scheduler that gradually adds Gaussian noise to images and uses a U-Net to predict and remove that noise. The U-Net uses timestep and class embeddings, allowing the model to generate images based on both the diffusion step and a specified class label. The DDPM implementation trains by predicting added noise and generates new images by repeatedly denoising random noise. Several VAE, DDIM, and latent diffusion sections are still incomplete and marked with TODOs
